5. Preparing the back matter 

  • Consult Sample O.N.E Author ODPub  to structure the Back matter

  • The back matter includes annexes and a glossary.

5.1. Annex document 

Tip 

  • References and notes of the annex are inserted at the end of each annex.

  • You should split the annexes if you use the Reference Management of Word and name the files (see naming convention ):

    • 99-backmatter-A

    • 99-backmatter-B

    • etc.

5.2. Glossary and Definitions 

A glossary is an alphabetical list of technical and unfamiliar terms, with explanations and definitions. A glossary is more detailed than a simple list of abbreviations and acronyms (in the front matter), and it should be placed towards the end of the book.

Tip 

For the Glossary use the Title, Title 2, Title 3 styles.

Once the back matter is finalised, run the Quality Checks  on each File and correct the errors. Check and clean the styles according the Section 1.9.
